# Projeto de Página de Notícias

Este é um projeto simples de uma página de notícias, criado como um exercício de desenvolvimento web para iniciantes. O objetivo é demonstrar a estrutura básica de uma página da web com HTML, estilização com CSS e um pouco de interatividade com JavaScript.

## Tecnologias Utilizadas

- **HTML5:** Usado para estruturar o conteúdo da página, como os artigos de notícias, o cabeçalho e o rodapé.
- **CSS3:** Usado para estilizar a página, incluindo o layout, cores, fontes e a responsividade para diferentes tamanhos de tela (como celulares e desktops).
- **JavaScript:** Utilizado para adicionar interatividade à página, como o menu de navegação que abre e fecha em dispositivos móveis.

## Como Visualizar o Projeto

Para visualizar este projeto, basta abrir o arquivo `index.html` em seu navegador de internet de preferência (como Google Chrome, Firefox, etc.). Não é necessário instalar nada.

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

- `index.html`: O arquivo principal que contém toda a estrutura da página.
- `README.md`: Este arquivo, que explica o projeto.
- `src/`: Uma pasta que contém todos os arquivos de suporte do projeto.
  - `css/`: Contém os arquivos de estilo.
    - `reset.css`: Usado para remover os estilos padrão do navegador.
    - `style.css`: O arquivo principal de estilos da página.
    - `variables.css`: Define as cores e outras variáveis usadas no CSS.
    - `responsive.css`: Contém os estilos para tornar a página responsiva a diferentes tamanhos de tela.
  - `images/`: Contém todas as imagens usadas na página.
  - `js/`: Contém o arquivo de script.
    - `script.js`: Onde está o código JavaScript para o menu interativo.
